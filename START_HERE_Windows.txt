=====================================================
 agentic-ai - How to run it on Windows (START HERE)
=====================================================

This runs an AI on your own computer using a free tool called
"Ollama" (it makes the AI fast). The run_windows.bat file sets up
everything for you - you mostly just double-click it.

The FIRST time, it may install a couple of things and ask you to
double-click it again. That's normal - just follow the messages in
the black window.


-----------------------------------------------------
STEP 1 - Install Python (one time)
-----------------------------------------------------
1. Go to:  https://www.python.org/downloads/
2. Click the big yellow "Download Python" button.
3. Open the file that downloads.
4. VERY IMPORTANT: on the first installer screen, tick the box at
   the bottom that says "Add python.exe to PATH".
5. Click "Install Now" and let it finish.


-----------------------------------------------------
STEP 2 - Get the code (one time)
-----------------------------------------------------
If you're reading this file on your computer, you already did this.
If not:
1. On the project's GitHub page, click the green "Code" button.
2. Click "Download ZIP".
3. Open your Downloads folder, right-click the ZIP, choose
   "Extract All...", and pick somewhere easy (your Desktop is fine).


-----------------------------------------------------
STEP 3 - Run it (double-click)
-----------------------------------------------------
1. Open the extracted folder (the one this file is in).
2. Double-click:  run_windows.bat
3. If Windows shows a blue "Windows protected your PC" box, click
   "More info", then "Run anyway". (This is normal.)

What happens the first time:
 - If Ollama (the AI engine) isn't installed yet, the window installs
   it - or opens its download page. If a page opens, install Ollama,
   then double-click run_windows.bat again.
 - It downloads a small AI model (about 1 GB) - you'll see progress.
 - It sets up the app.
 - When it's ready, your web browser opens by itself with a chat box.
   Type a message, press Enter, and you're talking to your own AI.

You may need to double-click run_windows.bat two or three times during
that first setup (once after installing Python, once after installing
Ollama). After that, it starts right up.

To STOP it: close the black window.
To START it again another day: double-click run_windows.bat.


-----------------------------------------------------
Make it faster or smarter
-----------------------------------------------------
Right-click run_windows.bat -> Edit (opens in Notepad) and find the
line with:  qwen2.5:1.5b
 - FASTER (a little less smart):  change it to  qwen2.5:0.5b
 - SMARTER (needs a strong PC):   change it to  qwen2.5:7b
Save, close Notepad, then run it again.
